# MLB 룰 스모크 테스트 체크리스트

## 정규시즌
- 새 시즌 시작 후 트레이드 마감일 표시가 현재 라운드 기준으로 보인다.
- 트레이드 마감일 이후 TRADE 탭과 제안 버튼이 비활성화된다.
- 라운드가 정상적으로 진행되고 스탯/순위가 갱신된다.
- HOME 화면 요약(시즌/기록/다음 경기)이 정상 표시된다.

## 부상/IL
- 부상 발생 시 선수 모달에서 MOVE TO IL이 보인다.
- IL 탭에 이동된 선수가 표시된다.
- 부상 회복 후 ACTIVATE로 복귀할 수 있다.
- IL60일 선수는 40인 로스터에서 제외된다.

## 40인 로스터
- FA/드래프트 영입 시 40인 초과하면 경고가 뜬다.
- 40-MAN 탭에서 명시적 40인 리스트가 표시된다.
- IL60은 40인에서 제외되고, IL10은 40인에 포함된다.

## 옵션/웨이버
- 옵션이 남아있는 선수는 AAA 이동 시 옵션이 1 감소한다.
- 옵션이 없는 선수는 웨이버로 이동한다.
- WAIVERS 탭에서 선수 확인 및 CLAIM 가능하다.
- 클레임 성공 시 내 로스터에 합류한다.
- TRANSACTIONS 패널에 옵션/웨이버/클레임 기록이 남는다.

## 일정/로그
- 스케줄 결과(FINAL) 클릭 시 상세 로그 모달이 열린다.
- 모달에서 MATCH 로그 이동 버튼이 동작한다.

## 포스트시즌
- 정규시즌 종료 후 포스트시즌 브래킷이 표시된다.
- POSTSEASON GAME 버튼으로 경기 진행이 가능하다.
- 시리즈 승리 시 다음 라운드로 진출한다.
- 우승팀 확정 후 오프시즌으로 전환된다.
